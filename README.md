#POBG
鸡地求生

简介
=====================
《鸡地求生》
模拟了《绝地求生》里面的小游戏POBG《鸡地求生》，
玩家通过击杀怪物得取积分，最终将与AI一起进行积分排名。

开发环境
=====================
操作系统：Windows
开发平台：Visual Studio 2019

开始游戏
=====================
点开tj-eatchicken.exe,点击Click To Change更换形象，
点击START GAME开始游戏。

操作说明
=====================
上：W
下：S
左：A
右：D
拾取道具：靠近道具自动拾取
装备手枪：按”1“”2“键装备并且更换手枪
放出子弹：鼠标点击需要放出子弹的方向

基础功能
======================
1.逃走躲避怪物袭击和boss发出的炮弹
2.攻击怪物获取积分
3.拾取枪具和子弹作为攻击道具
4.拾取医疗包补充血量
5.草丛中可以隐藏人物
6.人物和怪物都会被随机障碍阻挡
7.与AI对战并最终排名
